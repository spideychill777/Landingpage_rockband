# 🎸 Landing Page – Rock Band 

A simple and responsive website builted to show how the RockBand Shattered RiffsRiffs is the best band on the world engaging the attentions and prospecting futures concerts.

## Technologies Used
*   **HTML5**: Semantic structure.
*   **CSS3**: Styling, layout (Flexbox/Grid), and responsiveness.

## 🧱 Project Structure

```
├── assets                                 # Images, fonts, or icons
│   ├── css
│   │   └── style.css                      # Stylesheet
│   ├── icons
│   │   └── logo.ico
│   ├── img
│   │   ├── img-1.jpg ... img-10.jpg
│   │   └── guide.png
│   └── js
│       └── main.js
├── index.html                                # Main HTML file
└── README.md                                 # Project documentation
```



## 🚀 How to Run

1.  **Clone or download** this repository to your local machine.
2.  Open the project folder.
3.  Locate the `index.html` file.
4.  **Double-click** `index.html` to open it in your default web browser.

Alternatively, you can use a live server extension in VS Code to view the project.
